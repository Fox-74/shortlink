1. Присутствует ли в БД ссылка, которую вводил пользователь. Если присутствует, получить из базы данных ключ короткой ссылки, выполнить пункту 5. Иначе продолжить следующие шаги.

2. В БД такого хэша нет, создать новый хэш определённой длины.

3. Найти в БД созданный хэш. Если найден, то вернуться к пункту 2 и создать новый хэш. Иначе, дальше.

4. Если хэш в БД отсутствует, записать в БД хэш с длинной ссылкой.

5. Вывести на экран ключ ссылки.
